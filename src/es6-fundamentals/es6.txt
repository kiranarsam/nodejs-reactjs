
What is ES6?
ES6 stands for ECMAScript 6.

ECMAScript was created to standardize JavaScript, and ES6 is the 6th version of ECMAScript,
It is published in 2015, known as ECMAScript 2015.


Why Should I Learn ES6?
React uses ES6, need to familiar with new features like:
1. classes
2. Arrow Functions
3. Variables (let, const, var)
4. Array Methods like .map(), etc
5. Destructuring
6. Modules
7. Ternary Operator
8. Spread Operator
